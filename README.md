# Email Address Generator (Python)

This is my first Python learning project.  
The program generates possible email address formats based on a user's:

- First name
- Last name
- Company name

## How it works

The script asks the user to enter their first name, last name, and company name.  
Then it automatically generates several common email address formats such as:

- first.last@company.com
- f.last@company.com
- last.f@company.com

## Technologies used

- Python
- String manipulation
- User input handling
