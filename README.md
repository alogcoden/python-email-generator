# Email Address Generator (Python)

A small educational Python project that generates common corporate email address formats based on a user's name and company.

## 🚀 Live Demo

👉 **[Open Live Demo](https://alogcoden.github.io/python-email-generator/)**

## What this project does

The Python script asks the user to enter:
- First name
- Last name
- Company name

Then it generates several email address options in popular formats:
- `first.last@company.com`
- `f.last@company.com`
- `last.f@company.com`

All input values are converted to lowercase to keep the output consistent.
